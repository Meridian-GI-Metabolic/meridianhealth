How to Use This Schedule

Each of the ten posts in this plan runs through three campaign cycles. Each cycle corresponds to one full publishing round of the post. Allow at least 7 days of live data per cycle before recording results and rotating to the next variant. Record metrics in the results tracker at the close of each cycle before publishing the next.

Cycle What to Run What to Decide
Cycle 1 Recommended hook variant (see per-post column). Short Caption on Instagram. Long Caption on LinkedIn. Record the key metric listed for the post. Note whether the result meets, exceeds, or falls below benchmark.
Cycle 2 Next hook variant (see per-post column). Reverse the caption assignment: Long Caption on Instagram. Short Caption on LinkedIn. Compare the same metric to Cycle 1. Identify the stronger hook. Note whether caption reversal changed the result on either platform.
Cycle 3 Confirmed best hook from Cycles 1 and 2 (or Cycle 3 variant if noted). Winning caption length locked in for each platform. Record final metric. Confirm house format for this post going forward. Archive results in the tracker.

Cycle Decision Rules

Between Cycle 1 and Cycle 2: Record the primary metric in the results tracker before publishing the Cycle 2 version. Do not rely on memory. The rotation only produces usable data if results are logged at the close of each cycle, not retrospectively.

Between Cycle 2 and Cycle 3: Compare the Cycle 1 and Cycle 2 metric values. Select the stronger hook. If the difference is less than 15% (relative to the higher value), treat the hooks as equivalent and apply the tiebreaker noted in the per-post Cycle 3 row above.

Caption length decision: At the close of Cycle 2, compare platform performance under each caption assignment. If Short on Instagram outperformed Long on Instagram in both cycles, Short becomes the Instagram standard for that post. If the reversal in Cycle 2 produced a meaningfully better result, that length becomes the standard.

After Cycle 3: The winning hook and caption length for each post are confirmed as the house format for that post going forward. Archive all three results in the results tracker before closing the cycle. Do not run a fourth cycle unless a post is being materially reedited or relaunched for a new campaign.
